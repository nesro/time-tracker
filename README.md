time-tracker
============

Very simple time tracker

Usage:
./tt -w your_action

and when finished, close the terminal or Ctrl+C

version 1.0.0

Example:

``n@t:~/Dropbox/time-tracker$ ./tt -w BI-SZZ
action: BI-SZZ
today: none
since: 17:30:04
elapsed: 00:00:07^C
This track has been logged!

n@t:~/Dropbox/time-tracker$ ./tt -w BI-SZZ
action: BI-SZZ
today: 00:00:07
since: 17:30:13
elapsed: 00:01:21^C
This track has been logged!``
